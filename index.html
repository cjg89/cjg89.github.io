Events at UCF

Complete rebuild of the University of Central Florida online events system using Python and Django

UCF's events system was completely rebuilt from scratch to improve the overall user experience, SEO, and to allow for a more flexible, extensible codebase. The updated system runs on the Django framework.

The UCF community is capable of logging into the system and creating events for their own calendars, and/or for submission to UCF's main calendar for greater public exposure. Events can be scheduled on a recurring basis (i.e. daily/weekly/monthly) and can be copied to other calendars. Entire calendars can also be subscribed to.

The entire system is built to take advantage of a cache layer (Varnish). A Russian-doll object methodology allows for granular caching of individual pieces of content, rather than entire pages.

The site is built on the Twitter Bootstrap 3 CSS framework and is responsive on both the frontend and manager interface.

Weather Data Feed

A JSON feed that provides simple weather data from NOAA

This feed was built to provide a simple set of weather data for use in other UCF-related applications, but can be easily customized for other locations. It provides temperature, condition information, and a set of weather icons for the current time, today, and the next five days, simplifying the complex set of returned data from NOAA. NOAA's weather status codes are translated to the more commonly-used Yahoo Weather status codes.

A simple caching mechanism is built in to prevent excessive requests to the NOAA service. New data is fetched only when the feed is requested and the existing saved data has gone stale.

Skills Contributed:

  • PHP
  • JSON
  • 3rd Party API (NOAA)
